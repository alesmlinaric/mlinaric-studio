# Mlinaric Studio

Personal portfolio website built to present selected web development, design and client projects.

## Overview

Mlinaric Studio is my personal portfolio website, created to showcase my work as a junior web developer. The website presents selected projects, case studies, services, skills and contact information in a clean and responsive layout.

The goal of the project is to create a professional online presence that connects visual design, front-end development and real-world client work.

## Live Website

https://mlinaric.studio

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion
* React Router

## Features

* Responsive multi-page website
* Project overview and case study pages
* Modern UI with subtle animations
* SEO-friendly structure
* Contact page
* Clean component-based architecture

## Pages

* **Home** – landing page with hero section, featured projects, services, process, skills and call-to-action
* **Projects** – overview of selected portfolio projects
* **Project Detail** – individual case study pages with project overview, challenge, solution, tech stack and results
* **About** – personal introduction and background
* **Contact** – contact information and contact form

## Project Structure

```txt
src/
├── components/      # Reusable UI components
├── data/            # Project and website data
├── pages/           # Page components
├── App.tsx          # Main app component
├── main.tsx         # Entry point
└── index.css        # Global styles
```

## What I Focused On

* Building a clean and professional portfolio structure
* Creating reusable React components
* Designing responsive layouts with Tailwind CSS
* Presenting real client work through case studies
* Improving visual hierarchy, spacing and user experience
* Using Framer Motion for subtle page and section animations

## Purpose

This project is part of my professional portfolio and is used when applying for junior web developer / front-end developer roles.

## Local Development

```bash
npm install
npm run dev
```

To create a production build:

```bash
npm run build
```
